# sounds-like-skill-issue

A collection of Claude Code plugins that work universally across projects.

## Purpose

This repo stores my personal Claude Code plugins — skills, hooks, slash commands, and other extensions — that are general-purpose enough to apply to any codebase, not tied to a specific project.

## Structure

Each plugin lives in its own directory.

## Usage

Reference plugins from this repo in your Claude Code configuration (`~/.claude/settings.json` or per-project `.claude/settings.json`).
